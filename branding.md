# PayOrbit — Brand Guidelines

> Distilled from the PayOrbit Brand Identity deck (2026). Source of truth for logo, color, typography, and asset usage across product UI, marketing, and social.

---

## 1. Brand Overview

PayOrbit is a next-generation cross-border payroll infrastructure. It enables companies, startups, DAOs, NGOs, and global teams to pay employees and contractors anywhere in the world using stablecoins (USDC) and Stellar-based assets.

**Brand message:** Global payroll shouldn't take days. PayOrbit enables companies to pay global teams instantly using stablecoins on Stellar. No delays. No borders. No high fees. Payroll, reimagined.

**Core value props:** instant global payroll settlements, low-cost cross-border transactions, bulk payroll processing, real-time payment tracking, automated payroll scheduling, multi-currency support — powered by blockchain infrastructure instead of SWIFT-era banking rails.

**Brand tone:** bold, modern, built for the future of finance. Authoritative in the product, approachable in the brand.

---

## 2. Logo

### Concept
The mark began as a 3D orbital form — three intersecting rings around a central sphere — representing money in constant motion, orbiting across borders without friction. The rings represent the intersecting layers of the PayOrbit infrastructure: the sender, the Stellar network, and the recipient, orbiting a central point of trust. The 3D concept was distilled into a flat symbol that scales from app icons to large-format print.

### Wordmark
The wordmark (orbit symbol + typography) is the primary identifier of the brand, communicating speed, trust, and global reach. A monochrome variant exists for light backgrounds, print, and single-color applications.

### Logo variants
| Variant | Usage |
|---|---|
| **Purple background** (primary) | All main brand touchpoints — website headers, app interfaces, marketing materials where brand purple dominates |
| **Dark / black background** | Dark mode logo — dark UI surfaces, pitch decks, premium contexts needing contrast without color |
| **Light / grey background** (monochrome) | Light backgrounds, print, documents, limited color reproduction contexts |

### Icon variants
- **White-on-black** — default app icon
- **Monochrome dark** — light surfaces and single-color print
- **Magenta / purple-to-pink gradient accents** — campaigns, social media, promotional material
- **Extended palette variants** — co-branded contexts, app UI states, future product lines; all retain the orbital form

## 3. Color

### Primary — Electric Purple `#5501FF`
Anchors all main brand touchpoints.

| Token | Hex | Usage |
|---|---|---|
| `purple-100` | `#5501FF` | 100% — backgrounds, CTAs, hero sections |
| `purple-mid` | `#4600D2` | UI components, hover states, layered elements |
| `purple-deep` | `#3E02B7` | Shadows, pressed states, dark overlays |

### Accent — Cyan `#00FFBB`
Represents speed and instant transaction flow. Use sparingly.

| Token | Hex | Usage |
|---|---|---|
| `cyan-100` | `#00FFBB` | Highlights, success states, action confirmation |
| `cyan-mid` | `#A9FFE8` | Backgrounds, card surfaces |
| `cyan-light` | `#E3FFF8` | Notification states, subtle fills |

### Energy — Magenta `#FF00A6`
Campaign/social color; pairs with purple for the signature duotone.

| Token | Hex | Usage |
|---|---|---|
| `magenta-100` | `#FF00A6` | Campaign graphics, icon variants, gradient elements |
| `magenta-mid` | `#FF4DC1` | Notification badges, tag backgrounds |
| `magenta-light` | `#FFB2E4` | Secondary button fills, soft UI |

### Secondary palettes
| Token | Hex | Usage |
|---|---|---|
| `blue` (Sec-1) | `#00BFFF` | Links, active states, progress indicators, data viz, informational states where purple feels too dominant |
| `near-black` (Sec-2) | `#1F1F1F` | Dark mode primary background, pitch decks; grey scale from here upward for typography, dividers, card backgrounds, surface layering |
| `off-white` (Sec-3) | `#F3F3F3` | Default light-mode background, documents, print; light blue-grey tints bridge white → electric blue for modals, tooltips, inputs |

### Gradients
| Name | Stops | Usage |
|---|---|---|
| **PayOrbit 1** (signature) | `#5501FF` → `#FF00A6` | Hero sections, social visuals, orbital icon accent variants |
| **PayOrbit 2** | adds `#00FFBB` to the spectrum | Product UI backgrounds, onboarding screens, premium feature highlights |

Gradients communicate motion, energy, and global reach — used in hero visuals, 3D assets, and marketing.

### Principles
- High-contrast palette by design — visibility across all screen sizes and backgrounds.
- Neutrals (`#1F1F1F`, `#F3F3F3`) anchor the system and give vibrant colors room to breathe; they ensure accessibility across UI and marketing.
- Inspiration: iridescent 3D surfaces, liquid metal / holographic textures, saturated fabric depth, fluid electric-blue motion.

## 4. Typography

### Primary — Inter
Chosen for exceptional legibility across digital interfaces; clean, technical precision that communicates trust and clarity for a financial product. Applied across all UI components, headings, body copy, and data displays in the product.

| Weight | Usage |
|---|---|
| Light | Supporting text, captions, secondary info |
| Regular | Body copy, interface labels, instructional text (default) |
| Medium | Subheadings, card titles, emphasized UI elements |
| Semi-Bold | Primary navigation, section headers, CTAs |
| Bold | Hero headings, campaign statements, high-impact moments |

### Secondary — DM Sans
Rounded, friendly geometry that softens the technical edge. Used for marketing communications, onboarding flows, social media graphics, promotional materials, and consumer-facing content.

| Weight | Usage |
|---|---|
| Light | Large-format marketing/display text |
| Regular | Social captions, email body copy, campaign descriptions |
| Medium | Pull quotes, feature callouts, mid-weight marketing headlines |
| Bold | Hero statements, event graphics, maximum-impact moments |

### Pairing rules
- **Inter = structure and precision (product). DM Sans = approachability and warmth (brand).**
- Heading and body styles use fixed size ratios — apply type consistently without guesswork.
- Numerals use tabular spacing for financial contexts (transaction amounts, account numbers, data tables align cleanly).
- Full Latin glyph coverage incl. currency symbols and special characters.
- The pairing scales from 400px mobile screens to large-format billboards.

## 5. 3D Orbital Asset & Pattern

- The 3D orbital asset is the visual centerpiece — money in frictionless orbit across borders, currencies, and time zones.
- Large-scale form: hero sections, campaign visuals, pitch decks, premium moments.
- Compact form: app icons, social profile visuals, favicons — same iridescent finish and orbital structure at any size.
- Iridescent finish (electric purple / cyan / rose) mirrors the gradient system, tying the mark to its 3D expression.
- The orbital appears in all social media templates as a visual anchor.

---

## 6. Social Media

- **Default canvas: purple brand background** — instant recognition in feeds dominated by white/grey.
- **Electric blue variant** — reserved for product launch campaigns, feature announcements, and awareness content; never replaces purple as the default.
- Content hierarchy (both variants): bold question-led headline → concise brand statement → direct CTA (dual app store badges).
- Stellar blockchain education is a key content pillar (speed, cost, trust of the underlying infrastructure).
- Every post carries the PayOrbit logo and brand typefaces.
- Templates live in the shared Figma library with auto-layout enabled.

---

## 7. Target Persona (reference)

Head of Operations at a seed-stage, remote-first SaaS startup (e.g., "Amara Kone" / "James Osei" personas) — responsible for paying a globally distributed team, frustrated by slow, expensive cross-border payroll.
